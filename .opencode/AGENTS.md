# CloudOS OpenCode Agents

This directory defines specialized AI agents for CloudOS development.

- `@backend` ‚Äî Go backend services, core APIs
- `@frontend` ‚Äî React/TypeScript dashboard and UI
- `@database` ‚Äî Schema design, migrations, queries
- `@architect` ‚Äî System design and architecture decisions
- `@reviewer` ‚Äî Code review for all layers
- `@devops` ‚Äî Docker, CI/CD, deployment configurations
- `@testing` ‚Äî Test strategy and implementation
- `@docs` ‚Äî Documentation generation and review
- `@security` ‚Äî Security audits and vulnerability scanning
- `@mobile` ‚Äî React Native mobile app
- `@desktop` ‚Äî Tauri desktop app
- `@plugin` ‚Äî Plugin SDK and plugin development
- `@cli` ‚Äî CLI tool development
- `@ai-ml` ‚Äî AI/ML integration and model deployment
- `@integration` ‚Äî Third-party integrations and plugins

## Usage

Reference agents in prompts using `@agent-name` to delegate specific tasks.
